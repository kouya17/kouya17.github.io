Setting i2c#

Enabling i2c#

  1. sudo raspi-config
  2. select 5 Interfacing Options
  3. select P5 I2C
  4. select Yes

Installing packages#

sudo apt-get update
sudo apt-get install i2c-tools python-smbus

Connect to OLED#

Wiring#

  1. confirm pin assign
    • http://xn–ccke1di9d4h.com/blog/raspberrypi-zero-wh-pin/
  2. connect 3.3V, GND, SCL, SDA

Check i2c#

  1. confirm slave address by sudo i2cdetect -y 1
